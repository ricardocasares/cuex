# cuex

Currency exchanger

## Setup

Clone the repo,

```sh
git clone https://github.com/ricardocasares/cuex.git
```

Install `npm` dependencies,

```sh
npm i
```

Configure your API endpoint inside `.env` file

```
API=https://api.exchangeratesapi.io/latest
```

Start development server

```sh
npm run dev
```
